cycle gan 문제점 :
1. 1:1 매칭밖에 안됨 --> 도메인 간 관계 복잡해지면 도메인 간 관계를 제대로 파악 X
2. 인스턴스 모양 변경 잘 안됨
3. 배경 제외 인스턴스만 깨끗하게 변경안됨
4. 인스턴스 여러개일 경우 인스턴스끼리 통합되어 변환 --> 불확실 변환


cartoon gan :
만화의 특징인 부드러운 색감과 날카로운 edge를 가진 만화의 특징을 가지도록 전처리 방식, loss function 변화


insta gan : 
cycle gan 은 배경을 제외한 인스턴스 형태를 잘 바꾸지 못하고 GPU 메모리도 많이 들기 때문에
loss function 변경, mini-batch 학습법 도입


large-scale gan : 
이미지 to 이미지 변환시 해상도가 떨어지기 때문에 배치크기와 하이퍼 파라미터 갯수를 늘려 해상도 좋게 함
